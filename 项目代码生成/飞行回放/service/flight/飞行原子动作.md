### 飞行回放原子动作体系 · 100项

**四大维度标注：** `[操纵]` `[姿态]` `[轨迹]` `[状态]`

------

#### 一、俯仰类（Pitch） 1–13

1. **俯仰角稳定增大** `[姿态]` — 俯仰角θ以 2°~15°/s 单调递增，持续 ≥0.5s，无反向，初始θ任意
2. **俯仰角稳定减小** `[姿态]` — θ以 2°~15°/s 单调递减，持续 ≥0.5s，无反向
3. **俯仰角急剧增大** `[姿态]` — θ变化率 >15°/s，持续 ≥0.3s，通常对应大拉杆输入
4. **俯仰角急剧减小** `[姿态]` — θ变化率 <−15°/s，持续 ≥0.3s
5. **俯仰角正值保持** `[姿态]` — 5° ≤ θ ≤ 30°，偏差 <±1°，持续 ≥2s
6. **俯仰角零值保持（水平）** `[姿态]` — −1° ≤ θ ≤ +1°，持续 ≥2s
7. **俯仰角负值保持（低头）** `[姿态]` — −30° ≤ θ ≤ −5°，偏差 <±1°，持续 ≥2s
8. **大仰角保持** `[姿态]` — 30° < θ ≤ 90°，偏差 <±2°，持续 ≥1s，速度显著衰减
9. **大俯角保持** `[姿态]` — −90° ≤ θ < −30°，偏差 <±2°，持续 ≥1s，速度显著增加
10. **俯仰角穿越零度（仰→俯）** `[状态]` — θ由正值穿越0°至负值，穿越速率 2°~30°/s
11. **俯仰角穿越零度（俯→仰）** `[状态]` — θ由负值穿越0°至正值，穿越速率 2°~30°/s
12. **俯仰角穿越+90°（过顶）** `[状态]` — θ越过+90°继续向后，穿越速率 5°~60°/s，为边界事件
13. **俯仰振荡（PIO特征）** `[姿态]` — θ以 1~3Hz 频率周期性正负交替，峰值 ±3°~±15°，持续 ≥3个周期

#### 二、滚转类（Roll） 14–24

1. **左滚转增速** `[姿态]` — 坡度φ向负方向变化，滚转角速率 p 为 5°~180°/s，单调递增中
2. **右滚转增速** `[姿态]` — φ向正方向变化，p 为 5°~180°/s，单调递增中
3. **左坡度保持（小）** `[姿态]` — −30° ≤ φ ≤ −5°，偏差 <±1°，持续 ≥2s
4. **右坡度保持（小）** `[姿态]` — 5° ≤ φ ≤ 30°，偏差 <±1°，持续 ≥2s
5. **大坡度保持（左，≥60°）** `[姿态]` — −90° ≤ φ ≤ −60°，偏差 <±2°，持续 ≥1s，需持续拉杆
6. **大坡度保持（右，≥60°）** `[姿态]` — 60° ≤ φ ≤ 90°，偏差 <±2°，持续 ≥1s
7. **翼平保持** `[姿态]` — |φ| ≤ 2°，持续 ≥3s，协调飞行
8. **坡度穿越90°（左）** `[状态]` — φ从 −60°~−89° 穿越 −90°，穿越速率 10°~120°/s
9. **坡度穿越90°（右）** `[状态]` — φ从 60°~89° 穿越 +90°，穿越速率 10°~120°/s
10. **坡度穿越180°（进入倒飞）** `[状态]` — |φ| 穿越 180°，穿越速率 10°~180°/s，倒飞状态建立
11. **高速连续滚转** `[姿态]` — 滚转角速率 p ≥ 180°/s，持续 ≥360°，单轴主导，俯仰/偏航输入 <5°

------

#### 三、航向/偏航类（Yaw/Heading） 25–33

1. **航向角持续左转** `[姿态]` — ψ以 1°~10°/s 单调减小，持续 ≥2s，坡度介入但不为主导
2. **航向角持续右转** `[姿态]` — ψ以 1°~10°/s 单调增大，持续 ≥2s
3. **航向保持** `[姿态]` — |Δψ| ≤ 1°，持续 ≥5s，偏航率 |r| < 0.5°/s
4. **侧滑角增大（左，正β）** `[姿态]` — β 从 0° 增大至 5°~30°，增大率 1°~15°/s，非协调状态
5. **侧滑角增大（右，负β）** `[姿态]` — β 从 0° 减小至 −5°~−30°，变化率 1°~15°/s
6. **侧滑角归零（协调修正）** `[状态]` — |β| 从 >3° 减小至 <0.5°，修正速率 1°~10°/s
7. **偏航角速率保持（协调转弯）** `[姿态]` — 偏航率 r 在目标值 ±0.5°/s 内稳定，对应水平盘旋
8. **偏航振荡（荷兰滚）** `[姿态]` — β 和 φ 以 0.3~2Hz 耦合振荡，|β|峰值 2°~15°，持续 ≥3周期
9. **航向快速切换** `[状态]` — Δψ ≥ 30° 在 ≤3s 内完成，偏航率峰值 ≥15°/s，单次单方向

------

#### 四、轨迹角与爬降类（FPA / Climb / Descent） 34–44

1. **轨迹俯仰角（γ）稳定增大** `[轨迹]` — γ以 1°~8°/s 单调递增，持续 ≥1s，速度矢量上扬
2. **轨迹俯仰角（γ）稳定减小** `[轨迹]` — γ以 1°~8°/s 单调递减，持续 ≥1s
3. **正γ稳定保持（稳定爬升）** `[轨迹]` — 2° ≤ γ ≤ 20°，偏差 <±0.5°，持续 ≥5s，Vz > 0
4. **负γ稳定保持（稳定下降）** `[轨迹]` — −20° ≤ γ ≤ −2°，偏差 <±0.5°，持续 ≥5s，Vz < 0
5. **γ穿越零度（由降转升）** `[状态]` — γ从负值穿越 0°，穿越速率 1°~15°/s，轨迹最低点
6. **γ穿越零度（由升转降）** `[状态]` — γ从正值穿越 0°，穿越速率 1°~15°/s，轨迹最高点
7. **大仰角轨迹段（γ≥45°）** `[轨迹]` — γ ≥ 45°，持续 ≥0.5s，速度快速衰减 >5kn/s
8. **大俯角轨迹段（γ≤−45°）** `[轨迹]` — γ ≤ −45°，持续 ≥0.5s，速度快速增加 >5kn/s
9. **垂直爬升段（γ≥80°）** `[轨迹]` — 80° ≤ γ ≤ 90°，持续 ≥0.5s，为近垂直状态
10. **垂直俯冲段（γ≤−80°）** `[轨迹]` — −90° ≤ γ ≤ −80°，持续 ≥0.5s，速度增幅 >10kn/s
11. **倒飞水平轨迹保持** `[轨迹]` — |γ| ≤ 1°，|φ−180°| ≤ 3°，持续 ≥2s，负过载 −1g~−3g

------

#### 五、垂直速度类（Vertical Speed） 45–51

1. **垂直速度（Vz）持续增大（爬升率增加）** `[轨迹]` — Vz 从 <X 单调增至 X+200ft/min 以上，持续 ≥3s
2. **垂直速度（Vz）持续减小（下降率增加）** `[轨迹]` — Vz 从 >X 单调减至 X−200ft/min 以下，持续 ≥3s
3. **Vz 正值稳定保持** `[轨迹]` — 200 ≤ Vz ≤ 6000 ft/min，偏差 <±50 ft/min，持续 ≥5s
4. **Vz 负值稳定保持** `[轨迹]` — −6000 ≤ Vz ≤ −200 ft/min，偏差 <±50 ft/min，持续 ≥5s
5. **Vz 归零（平飞建立）** `[状态]` — |Vz| 从 >200 ft/min 减小至 <50 ft/min，过渡时间 ≥1s
6. **大下沉率段** `[轨迹]` — Vz ≤ −3000 ft/min，持续 ≥2s，通常对应大俯角或减速俯冲
7. **Vz 过零穿越（由正转负，爬转降）** `[状态]` — Vz 穿越 0，穿越速率 |dVz/dt| ≥ 50 ft/min/s

#### 六、空速类（Airspeed） 52–60

1. **空速持续增大（小加速）** `[轨迹]` — dV/dt 为 0.5~3 kn/s，持续 ≥3s，高度变化 <300ft
2. **空速持续增大（大加速）** `[轨迹]` — dV/dt ≥ 3 kn/s，持续 ≥2s，通常加力介入
3. **空速持续减小（小减速）** `[轨迹]` — dV/dt 为 −0.5~−3 kn/s，持续 ≥3s
4. **空速持续减小（大减速）** `[轨迹]` — dV/dt ≤ −3 kn/s，持续 ≥2s，通常减速板/收油
5. **空速稳定保持** `[轨迹]` — |ΔV| ≤ 2 kn，持续 ≥5s，推阻平衡
6. **近失速速度保持（1.1~1.3Vs）** `[状态]` — V 在 1.1~1.3 倍失速速度区间，持续 ≥2s，抖振可能存在
7. **超音速加速穿越（Ma 0.95→1.05）** `[状态]` — 在 ≤10s 内 Ma 穿越 1.0，激波结构突变
8. **跨音速减速穿越（Ma 1.05→0.95）** `[状态]` — Ma 从超音速回穿 1.0，操纵特性改变
9. **最大速度区间保持（Vmax±5kn）** `[状态]` — V 在 Vmax−5 ~ Vmax+5 kn，持续 ≥1s，极限边界

#### 七、油门/推力类（Throttle / Thrust） 61–68

1. **油门前推（慢推，小增推）** `[操纵]` — 推进速率 5°~15°/s（杆角），推力比从 X 增至 X+(5%~20%)，持续 ≥1s
2. **油门前推（快推，大增推）** `[操纵]` — 推进速率 >30°/s，推力比增量 >20%，持续 <2s
3. **油门后收（慢收）** `[操纵]` — 收油速率 5°~15°/s，推力比从 X 减至 X−(5%~20%)
4. **油门后收（快收，收至慢车）** `[操纵]` — 收油速率 >30°/s，推力比减至 ≤10% 慢车位，持续 <2s
5. **油门稳定保持** `[操纵]` — 油门位置偏差 <±1°（杆角），推力比变化 <±1%，持续 ≥5s
6. **加力接通** `[状态]` — 油门越过加力挡，发动机推力在 0.5~2s 内阶跃增大 ≥30%
7. **加力关断** `[状态]` — 退出加力挡，推力在 0.5~2s 内阶跃减小 ≥30%
8. **减速板展开** `[操纵]` — 减速板从 0% 展开至 50%~100%，展开时间 0.5~3s，阻力增大

#### 八、过载类（Load Factor） 69–76

1. **正过载稳定保持（1g）** `[姿态]` — 0.9g ≤ n ≤ 1.1g，持续 ≥3s，平稳非机动飞行
2. **正过载稳定保持（中等，2~4g）** `[姿态]` — 2g ≤ n ≤ 4g，偏差 <±0.3g，持续 ≥2s
3. **正过载稳定保持（大，≥5g）** `[姿态]` — n ≥ 5g，偏差 <±0.5g，持续 ≥1s
4. **正过载持续增大** `[姿态]` — dn/dt ≥ 0.5g/s，n 从 1g 以上单调递增，持续 ≥1s
5. **正过载持续减小** `[姿态]` — dn/dt ≤ −0.5g/s，n 单调递减，不进入负过载
6. **过载穿越零（1g→0g→负g）** `[状态]` — n 穿越 0g，穿越速率 |dn/dt| ≥ 0.3g/s
7. **负过载保持（倒飞）** `[姿态]` — −3g ≤ n ≤ −0.5g，持续 ≥1s，倒飞推杆维持
8. **峰值过载冲击（瞬态）** `[状态]` — n 在 ≤0.5s 内达到峰值后立即开始回落，峰值 ≥4g，持续 <0.5s

#### 九、迎角类（AoA） 77–83

1. **迎角持续增大** `[姿态]` — α 以 1°~10°/s 单调递增，持续 ≥0.5s，对应减速或拉杆
2. **迎角持续减小** `[姿态]` — α 以 1°~10°/s 单调递减，对应加速或推杆
3. **迎角稳定保持（小，2°~8°）** `[姿态]` — 2° ≤ α ≤ 8°，偏差 <±0.5°，持续 ≥3s，正常巡航范围
4. **迎角稳定保持（大，10°~25°）** `[姿态]` — 10° ≤ α ≤ 25°，偏差 <±1°，持续 ≥1s，大迎角机动
5. **迎角超过临界值（失速触发）** `[状态]` — α 超过 αcrit（通常 15°~28°，取决于构型），升力开始骤降，触发边界事件
6. **迎角快速回落（失速改出）** `[状态]` — α 从 >αcrit 以 >5°/s 回落至 αcrit−5° 以下，升力恢复
7. **近临界迎角保持（抖振区）** `[状态]` — αcrit−3° ≤ α ≤ αcrit，抖振振幅 >0.2g，持续 ≥1s

#### 十、舵面输入类（Control Surface Input） 84–91

1. **纵向杆前推施加** `[操纵]` — 纵杆从中立位前推，偏转量 5%~100% 行程，速率 10%~100%行程/s
2. **纵向杆后拉施加** `[操纵]` — 纵杆从中立位后拉，偏转量 5%~100% 行程
3. **纵向杆归中** `[操纵]` — 纵杆偏转量从 >10% 回至 <2%，归中速率 5%~100%行程/s
4. **横向杆左压施加** `[操纵]` — 横杆向左偏转 5%~100%，速率 10%~100%行程/s，单次施加
5. **横向杆右压施加** `[操纵]` — 横杆向右偏转 5%~100%，速率 10%~100%行程/s
6. **横向杆归中** `[操纵]` — 横杆从偏转位回至 <2%，归中速率 5%~100%行程/s
7. **左踩舵施加** `[操纵]` — 左舵踏板偏转 5%~100%，速率 10%~100%行程/s，产生左偏航力矩
8. **右踩舵施加** `[操纵]` — 右舵踏板偏转 5%~100%，速率 10%~100%行程/s

#### 十一、气动构型类（Aerodynamic Configuration） 92–96

1. **起落架收起** `[状态]` — 起落架从放下锁定→完全收起锁定，过渡时间 3~8s，阻力系数 ΔCd 减小 >30%
2. **起落架放下** `[状态]` — 起落架从收起→完全放下锁定，过渡时间 3~10s，VLO ≤ 250 kn
3. **襟翼增大一级** `[状态]` — 襟翼从当前档位偏转至下一档，偏转量 10°~20°，过渡时间 2~5s，升力增加
4. **襟翼收回一级** `[状态]` — 襟翼从当前档位减小至上一档，偏转量 −10°~−20°，过渡时间 2~5s
5. **减速板收起** `[操纵]` — 减速板从 >50% 完全收回至 0%，收起时间 0.5~2s

#### 十二、稳定修正与复位类（Stabilization & Recovery） 97–100

1. **高度偏差修正（高度捕获）** `[轨迹]` — Δh 从 >50ft 减小至 <10ft，修正速率 Vz 介入量 100~500 ft/min，持续 ≥5s，终态 |Vz| < 50 ft/min
2. **速度偏差修正（速度捕获）** `[轨迹]` — |ΔV| 从 >5 kn 减小至 <1 kn，修正周期 ≥3s，通过油门±5°~15° 调节实现
3. **航向偏差修正（航向捕获）** `[姿态]` — |Δψ| 从 3°~30° 减小至 <1°，通过坡度 5°~20° 介入，持续 ≥3s，终态 |r| < 0.3°/s
4. **三轴姿态同时归稳（复位）** `[状态]` — θ→目标±1°，φ→0°±2°，β→0°±0.5°，三者同时满足，归稳时间 ≥2s，为机动结束的终态判据
